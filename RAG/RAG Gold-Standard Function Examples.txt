Example 1 — Function Contract 
Function: chunk_text () 
 
Purpose : 
Split document text into retrieval-safe chunks . 
 
Inputs : - text: string - max_tokens: integer - overlap_tokens: integer - metadata: object 
 
Output : - list of chunks 
 
Each chunk must include : - chunk_id - text - token_count - metadata 
 
Rules : - Reject empty text . - Preserve metadata . - Do not silently drop oversized text . - Do not exceed max_tokens unless unavoidable .
